# Repo for cucumberway2automation
This is the repo for cucumber, this readme will explain step by step how to use and run your tests

## What is required
 Create the following packages on java class:

* Features: (right click on java drop-down, then select New > Package folder > capture Features on the package pop-up then click Enter)

* Pages: right click on java drop-down, then select New > > Package folder > capture Pages on the package pop-up then click Enter)

* Pages: right click on java drop-down, then select New > > Package folder > capture Runner on the package pop-up then click Enter)

* Pages: right click on java drop-down, then select New > > Package folder > capture StepDefinition on the package pop-up then click Enter)
